# 1 장 이맥스 기초

## 1. 표기 방식

### 이맥스 명령

이맥스의 명령은 컨트롤키나 ESC키와 같은 수정자와 뒤이어 나오는 문자로 이루어진다.

    C-g 컨트롤 키를 누른 상태에서 g를 누른다.
    ESC x ESC를 눌렀다 뗀 후에 x를 누른다.
    
## 2. 이맥스의 기초.

### 파일과 버퍼의 이해.

편집기는 실제 파일을 편집하지 않는다. 대신 파일의 내용을 임시 버퍼에 복사해서
편집한다.  버퍼는 파일처럼 보이지만 실제는 파일을 복사한 내용을 담고 있는
임시적인 작업공간이다. 

버퍼도 파일처럼 이름이 있다. 보통 버퍼명은 편집하고 있는 파일명과 같다. 어떤
버퍼에는 관련된 파일이 없다.  \*scratch\*는 연습장과 같은 임시 연습공간이며,
\*Help\*는 도움말을 보여주는 버퍼로 파일에 연결되어 있지 않다. 

파일 편집을 시작하면 파일을 버퍼로 복사한다. 파일을 편집하는 것처럼 보이지만
실제는 파일 자체가 아니라 버퍼를 고치는 것이다.

### 이맥스 편집 모드

이맥스는 다양한 종류의 편집모드를 제공한다. 모드라는 단어는 이맥스가 현재 작업에
적합한 환경으로 바뀌는 것을 의미한다. 글을 작성하기 위해서는 텍스트 모드(text
mode), 프로그램 작성을 위해서는 C 언어 모드가 있다. 

텍스트 모드와 C 언어 모드는 주모에 해당한다. 하나의 버퍼는 동시에 하나의
주모드에 있을 수 있으며, 주모를 빠져 나올 경우는 반드시 다른 모드로 들어가야만
한다. 

| 모드                       | 동작                                  |
|----------------------------|---------------------------------------|
| 기본모드(Fundamental mode) | 기본설정된 모드로 특별한 동작은 없다. |
| Text mode                  | 텍스트를 작성한다.                    |
| Mail mode                  | 메일을 작성한다.                      |
| RMAIL mode                 | 메일을 읽고 정리한다.                 |
| View mode                  | 파일을 보기만 하고 고치지 않는다.     |
| Shell mode                 | 이맥스에서 명령어를 실행한다.         |
| Ange-ftp mode              | 원격 시스템의 파일을 내받거나 본다.   |
| Outline mode               | 개요를 작성한다.                      |
| Telnet mode                | 원격시스템에 로그인한다.              |
| Indented text mode         | text mode 에서 자동으로 들여쓴다.     |
| Picture mode               | 간단한 선을 그린다.                   |
| Nroff mode                 | nroff의 파일을 편집한다.              |
| Tex mode                   | tex용 파일을 편집한다.                |
| LaTex mode                 | LaTex 파일을 편집한다.                |
| C mode                     | C 프로그램을 작성한다.                |
| C++ mode                   | C++ 프로그램을 작성한다.              |
| FORTRAN mode               | 포트란 프로그램을k작성한다.           |
| Emacs Lisp mode            | 이맥스의 LISP 함수를 작성한다.        |
| LISP mode                  | LISP 프로그램을 작성한다.             |
